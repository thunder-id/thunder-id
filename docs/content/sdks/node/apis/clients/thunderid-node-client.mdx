# `ThunderIDNodeClient`

`ThunderIDNodeClient` is an abstract base class that extends `ThunderIDJavaScriptClient` with Node.js-specific configuration. Use this class as the foundation when building a new framework-specific authentication client for a Node.js runtime. Examples include adapters for other Node.js server frameworks.

:::tip For Application Development
If you are integrating authentication into an application rather than building a framework SDK, use [`LegacyThunderIDNodeClient`](/docs/next/sdks/node/apis/clients/legacy-thunderid-node-client), which provides a complete, ready-to-use authentication API.
:::

## Overview

`ThunderIDNodeClient` inherits all methods from `ThunderIDJavaScriptClient` and applies the [`ThunderIDNodeConfig`](/docs/next/sdks/node/apis/config/thunderid-node-config) type by default. Because the class is abstract, you cannot instantiate it directly. Instead, extend it and implement any framework-specific initialization or middleware logic in the subclass.

## Usage

```ts title="src/MyFrameworkClient.ts" showLineNumbers
import { ThunderIDNodeClient, ThunderIDNodeConfig } from '@thunderid/node'

interface MyFrameworkConfig extends ThunderIDNodeConfig {
  customOption?: string
}

class MyFrameworkClient extends ThunderIDNodeClient<MyFrameworkConfig> {
  // Add framework-specific methods or override inherited ones here
}

export default MyFrameworkClient
```
