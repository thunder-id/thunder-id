# Node.js SDK

The ThunderID Node.js SDK (`@thunderid/node`) provides authentication and session management for server-side Node.js applications. The SDK implements the OAuth 2.0 Authorization Code flow with PKCE and supports multi-user session management.

## Installation

Install the <ProductName /> Node.js SDK using your preferred package manager:

<CodeGroup>
  <CodeBlock lang="bash" label="npm">
    npm install @thunderid/node
  </CodeBlock>

  <CodeBlock lang="bash" label="Yarn">
    yarn add @thunderid/node
  </CodeBlock>

  <CodeBlock lang="bash" label="pnpm">
    pnpm add @thunderid/node
  </CodeBlock>

  <CodeBlock lang="bash" label="Bun">
    bun add @thunderid/node
  </CodeBlock>
</CodeGroup>

## Features

### Clients

The SDK provides authentication clients designed for server-side Node.js applications.

- **[`LegacyThunderIDNodeClient`](/docs/next/sdks/node/apis/clients/legacy-thunderid-node-client)** — Full authentication client with sign-in, sign-out, token management, and session utilities
- **[`ThunderIDNodeClient`](/docs/next/sdks/node/apis/clients/thunderid-node-client)** — Abstract base class for building Node.js framework integrations

### Configuration

- **[`ThunderIDNodeConfig`](/docs/next/sdks/node/apis/config/thunderid-node-config)** — Configuration type for the Node.js SDK

### Utilities

Helper functions and constants for session and cookie management.

- **[`CookieConfig`](/docs/next/sdks/node/apis/utilities/cookie-config)** — Default cookie configuration constants
- **[`CookieOptions`](/docs/next/sdks/node/apis/utilities/cookie-options)** — Cookie configuration interface
- **[`generateSessionId()`](/docs/next/sdks/node/apis/utilities/generate-session-id)** — Generate unique session identifiers
- **[`getSessionCookieOptions()`](/docs/next/sdks/node/apis/utilities/get-session-cookie-options)** — Build cookie options with defaults applied
